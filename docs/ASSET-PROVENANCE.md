# Asset provenance

All visual assets are original and created for this repository. `assets/favicon.svg` is a geometric M/FC monogram. `assets/evidence-grid.svg` is an original editorial evidence matrix. Both are authored as repository SVG and contain no third-party photography, trademarks, or generated likenesses. IBM Plex is loaded from Google Fonts under its published open font license; system fallbacks remain usable if blocked. Screenshots are generated from the site by Playwright.
